# Aeivion Youtube downloader

**This app will help you download any youtube videos into your system without any waitings and delays you can see the downloading status in the terminal**

# Feature
- Download any youtube videos
- Maximum quality assured
- Directly connection with youtube server
- No delay like compression and stuffs ,there will delay for downloading the file whicb depends on the file size and internet connection.

# User Guide
- Extract the zip
- open terminal in the dir
- Execute the commands

# For Linux ( Ubuntu )

```bash
apt update 
apt upgrade 
pip install django yt-dlp
python manage.py runserver
```

# For Termux 

```bash
pkg update 
pkg upgrade 
pip install django yt-dlp
python manage.py runserver
```
# Fedora linux 

```bash
dnf update 
dnf upgrade 
pip install django yt-dlp
python manage.py runserver
```
